## 🎯 Objetivo
O principal objetivo deste repositório é armazenar e organizar os códigos e resoluções dos exercícios práticos desenvolvidos em sala, explorando os recursos do NumPy para análise e manipulação de dados, computação matemática e operações matriciais de alta performance no contexto de dados e inteligência de negócios.

## 🛠️ Tecnologias Utilizadas
- **[Python](https://www.python.org/)** - Linguagem de programação.
- **[NumPy](https://numpy.org/)** - Biblioteca fundamental para a computação científica no Python.

## 📂 Estrutura do Repositório
Os arquivos deste repositório representam o progresso nas aulas e resolução dos desafios passados em cada encontro.

- `funcoes.py` (e outros scripts a serem adicionados): Funções, lógicas em Python e soluções utilizando recursos do NumPy.

## 🚀 Como Executar
1. Certifique-se de ter o Python devidamente instalado na sua máquina.
2. No seu terminal, instale a biblioteca NumPy:
   ```bash
   pip install numpy
   ```
3. Execute localmente os scripts para testar as resoluções (em uma IDE ou no próprio terminal).
